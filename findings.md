# Research Findings — CausalCrisis V3

## Research Question

Can per-modality causal disentanglement with cross-modal causal fusion and hybrid ICA-adversarial learning achieve >90% weighted F1 on CrisisMMD multimodal crisis classification while generalizing to unseen disaster domains?

## Current Understanding

### Core Architecture Insight (Cycle 0 — Bootstrap)

CausalCrisis V3 is based on a **Structural Causal Model (SCM)** for multimodal crisis data, where observed features from images and text are generated by a mixture of **causal factors** (domain-invariant, predictive of the label) and **spurious factors** (domain-specific, correlated but not causally related to the label).

The key theoretical claim: disentangling these factors **per-modality** (rather than jointly as in CAMO) enables finer-grained control and better domain generalization because visual spurious factors (background, style) are fundamentally different from textual spurious factors (hashtags, locations).

### Literature-Informed Position

1. **CrisisSpot (90.9% F1)** — SOTA but uses BERT+ResNet (weaker encoders) and no causal theory
2. **CAMO** — Causal framework but VGG+BERT, joint disentangle, limited evaluation
3. **CCA (2025)** — ICA disentangle of CLIP features outperforms adversarial on 11 benchmarks → **key inspiration for hybrid approach**
4. **CIRL / InPer** — Validated causal intervention for single-modality DG → we extend to multimodal

### Gap We Fill
No existing work combines: (1) CLIP ViT-L/14 features + (2) per-modality ICA-adversarial disentanglement + (3) cross-modal causal factor C_vt + (4) backdoor adjustment — for crisis classification.

## Patterns and Insights

### From Literature Analysis (Pre-experimental)
- **CLIP is a strong starting point**: ViT-L/14 features are 768-d, well-aligned across modalities
- **ICA > Adversarial for CLIP**: CCA (2025) provides strong evidence
- **Per-modality > Joint disentangle**: Conceptually validated, needs empirical confirmation
- **C_vt is novel**: Cross-modal causal factor is our primary contribution

## Lessons and Constraints

### Known from V2 Experiments
- Adversarial training (GRL) requires careful lambda warmup — too aggressive = collapse
- Orthogonal loss alone is insufficient — causal and spurious features can still overlap
- Memory Bank size > 1000 is sufficient for BA; 2000 adds minimal benefit
- Training with 3 phases is complex but stabilizes convergence

### Known from Literature
- CCA achieves disentanglement without training instability (ICA is deterministic)
- SupCon loss significantly improves cross-domain transfer in representation learning
- Adaptive loss weighting eliminates hyperparameter sensitivity
